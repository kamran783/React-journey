CDN (Content Delivery Network)

Definition:
A CDN (Content Delivery Network) is a group of geographically distributed servers that deliver website content to users from the nearest server, instead of a single central server.

Why CDN is used

1.To increase website speed

2.To reduce latency (delay)

3.To reduce load on the main server

4.To improve reliability and security


How CDN works

1.Content is stored on a main (origin) server

2.CDN keeps cached copies of frequently used content on edge servers

3.When a user requests content, it is served from the nearest CDN server

4.If not available, CDN fetches it from the origin server and caches it
